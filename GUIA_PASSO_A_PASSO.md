# Guia rápido — PRENAT+ V5 Editor Fácil

## 1. Onde o aluno entra

Use o link normal do GitHub Pages:

`https://cursoprenat-sucesso.github.io/missao-ilhas-natureza-prenat/`

Esse é o link para colocar no Campo de Treino da plataforma.

## 2. Onde o professor entra

Use o mesmo link, mas com `/professor.html` no final:

`https://cursoprenat-sucesso.github.io/missao-ilhas-natureza-prenat/professor.html`

## 3. Como colocar subscrito

No enunciado, alternativa ou comentário, escreva normalmente e use os botões que aparecem embaixo do campo.

### Jeito 1: botão Subscrito

1. Escreva `H2O`.
2. Selecione apenas o número `2`.
3. Clique em `x2 Subscrito`.
4. Vai virar `H<sub>2</sub>O`.

### Jeito 2: botão Auto química

1. Escreva uma fórmula, por exemplo: `Na2CO3`.
2. Clique em `Auto química`.
3. O editor transforma em `Na<sub>2</sub>CO<sub>3</sub>`.

O texto aparece com tags dentro do painel, mas no jogo do aluno aparece formatado como Na₂CO₃.

## 4. Como colocar sobrescrito

1. Escreva `m/s2`.
2. Selecione o `2`.
3. Clique em `x2 Sobrescrito`.
4. Vai virar `m/s<sup>2</sup>`.

## 5. Como colocar fórmula

Clique em `Fórmula \( \)` e edite a fórmula gerada.

Exemplo:

`\( Q = m \cdot c \cdot \Delta T \)`

## 6. Como colocar imagem na questão

Você tem três caminhos:

### Caminho A: copiar e colar

1. Copie a imagem.
2. Clique na caixa: `Cole ou arraste a imagem aqui`.
3. Aperte `Ctrl + V`.
4. Veja se apareceu a prévia.
5. Salve a questão.

### Caminho B: escolher arquivo

1. Clique em `Escolher imagem`.
2. Selecione a imagem no computador.
3. Veja se apareceu a prévia.
4. Salve a questão.

### Caminho C: arrastar imagem

1. Arraste o arquivo da imagem para a caixa de imagem.
2. Solte dentro da caixa.
3. Veja se apareceu a prévia.
4. Salve a questão.

## 7. Como atualizar o jogo publicado

Depois de cadastrar as questões:

1. Vá na aba `4. Baixar arquivos`.
2. Baixe `questions.json`.
3. Se alterou fases/metas, baixe também `settings.json`.
4. No GitHub, clique em `Add file > Upload files`.
5. Envie os arquivos novos.
6. Clique em `Commit changes`.
7. Espere alguns minutos e teste o link do aluno.

## Observação importante sobre imagens coladas

Quando você cola imagem, ela fica salva dentro do `questions.json`. Isso facilita porque não precisa subir a imagem separada. Evite imagens enormes. A programação reduz a imagem automaticamente para ajudar o arquivo a não ficar pesado.
