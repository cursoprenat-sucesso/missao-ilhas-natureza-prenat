Como funciona

Este painel não altera o GitHub automaticamente. Ele gera os arquivos settings.json e questions.json. Depois você envia/substitui esses arquivos no GitHub.

O aluno acessa somente o link do jogo. O professor acessa este painel pelo link terminado em professor.html.

Fluxo exato

  1. Entre neste painel do professor.
  2. Configure a missão e as fases, se quiser alterar nomes, metas, vidas e quantidade de questões.
  3. Cadastre as questões.
  4. Baixe settings.json e questions.json.
  5. No GitHub, substitua os arquivos antigos pelos novos.
  6. O link do aluno atualiza sozinho em alguns minutos.

Imagem e fórmula

Para usar imagem em uma questão, envie a imagem para o GitHub junto com os arquivos e, no campo de imagem da questão, escreva apenas o nome do arquivo, por exemplo:

grafico-calorimetria.png

Para fórmulas, você pode usar LaTeX:

\( Q = m \cdot c \cdot \Delta T \)

Passo a passo no GitHub

  1. Volte ao repositório da missão.
  2. Clique em Add file > Upload files.
  3. Arraste os novos arquivos settings.json e/ou questions.json.
  4. Clique em Commit changes.
  5. Aguarde alguns minutos e teste o link do aluno.
